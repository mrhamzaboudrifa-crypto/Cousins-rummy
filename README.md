# Cousins Rummy Room — UI Prototype (Lobby + In‑Game)

This ZIP contains a **real HTML/CSS prototype** of the neon bar/lounge UI you approved.

## Files
- `index.html` — Lobby screen
- `game.html` — In‑game screen
- `styles.css` — Neon + glass styling
- `assets/` — Put your background image here

## Add your background image
Add a bar/loungy background image at:

`assets/bar-bg.jpg`

A portrait image around **1080×1920** looks best on phones.

## Open it
- Open `index.html` to view the lobby.
- Open `game.html` to view the in‑game layout.

## Publish on GitHub Pages
1. Upload the files to a GitHub repo (root).
2. GitHub → repo **Settings** → **Pages**
3. Source: Deploy from a branch → `main` / `(root)`
4. Visit:
   - Lobby: `https://YOURUSERNAME.github.io/REPO/`
   - Game: `https://YOURUSERNAME.github.io/REPO/game.html`
